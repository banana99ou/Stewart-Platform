# 춘계 항공우주학회 논문 발표 준비 계획서

### 논문 제목(안)

**스튜어트 플랫폼을 이용한 물리적 폐루프 HILS 기반 비행체 자세 제어 검증 시스템**

*A Physically Closed-Loop Hardware-in-the-Loop System Using a Stewart Platform for Vehicle Attitude Control Validation*

---

## 1. 연구 목표 요약

X-Plane 시뮬레이터와 연동된 스튜어트 플랫폼 위에 실제 FC(+IMU)를 탑재하여, **물리적 모션을 포함하는 폐루프 HILS 환경**을 구축하고 이를 통해 비행제어 성능을 검증.

구체적으로,

- X-Plane에서 계산된 항공기 자세를 스튜어트 플랫폼으로 구현하여 FC의 IMU에 물리적 모션 제공
- FC가 실제 IMU 신호를 이용해 제어입력을 생성하고 이를 다시 X-Plane에 적용하는 폐루프 구성
- 시뮬레이터 자세와 플랫폼/IMU 자세 간의 추종 오차(Mean, RMS, Max, Std) 정량화
- 오차 원인(지연, 기구학, 센서 노이즈, 좌표계 변환 등) 분석
- 제어 알고리즘 검증, 교육/훈련, 데모 플랫폼으로서의 활용 가능성 제시

---

## 2. 춘계학회 일정 기반 개발 및 실험 계획

춘계 항공우주학회 논문 초록 제출 기한이 **2월 13일**이므로, 여유를 두고 **2월 6일까지 초록 초안 제출** 할 수 있도록 하는 것을 1차 목표로 설정.

또한 발표 준비를 위해 **3월 20일까지 발표자료 초안**을 완성하는 것이 목표임.

### **(1) 1단계: 초록 제출을 위한 최소 실험 (1월 중순 ~ 2월 초)**

실제 물리적 폐루프 HILS임을 증명할 수 있는 1개 이상의 실험을 확보.

| 기간 | 목표 |
| --- | --- |
| ~ 1월 19일 | 스튜어트 플랫폼을 통해 FC IMU에 물리적 자세 자극 제공 |
| ~ 1월 20일 | FC가 X-Plane 항공기를 직접 제어하는 폐루프 구성 완료 |
| ~ 1월 27일 | 수평 비행(trim) 상태에서 안정적으로 자세 유지 가능 |
| ~ 1월 31일 | 0° → +5° 피치 명령에 대한 **Trim transition 실험 1회 이상 성공** |
| ~ 2월 3일 | X-Plane 자세 vs 플랫폼/IMU 자세 오차 그래프 산출 |
| **2월 6일** | 위 실험 결과를 바탕으로 초록 초안 작성 완료 |
