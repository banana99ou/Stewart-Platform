본 플랫폼 기반으로 발표를 하는 것은 문제 없고 좋은 것 같습니다. 발표는 플랫폼 제작 및 구현 자체에 초점을 두되, 다음 내용들이 구체적으로 제시되면 좋겠습니다. 

1. 플랫폼 제작 목적: X-Plane 시뮬레이터와 연동하여, 비행 중인 항공기나 드론의 실시간 자세를 화면을 통해서가 아닌 물리적 실체로 모니터링하기 위한 목적임.
관련해서 아래 내용을 구체화해야 할 것으로 생각됩니다:
  - 기존 HILS(Hardware-In-the-Loop Simulation) 대비 본 시스템의 차별점 명확화
    - 기존: FC의 센서 입력만 시뮬레이션 → 자세 정보는 디스플레이로만 확인
    - 본 시스템: 물리적 자세 구현 → FC의 IMU가 실제 운동을 감지 → 더 realistic한 closed-loop 테스트 가능
    - (한계) 자세각 범위 제한, 공기역학적 효과 부재 등으로 인해 기존 HILS를 완전히 대체할 수는 없고 보완적으로 활용 가능.
  - 활용 가치 제시
    - 비행 제어 알고리즘 검증 시 시각적 모니터링의 직관성 향상
    - 조종사 훈련, 데모, 교육용 활용 가능성
    - 실제 비행 테스트 전 preliminary validation 도구로서의 가치

2. 구체적인 활용 예시가 꼭 포함되면 좋겠습니다: 다음 중 선택하거나 조합:

(Option A) 트림점 천이 (Trim Transition)
  - 시나리오: 수평 직선 비행(level flight) → 상승/하강 천이 → 새로운 트림점 안정화
  - 관찰 포인트: 천이 과정에서의 피치각 변화, 스튜어트 플랫폼의 추종 성능
  - 정량적 결과: 시뮬레이터 자세 vs 플랫폼 구현 자세의 오차 그래프 제시

(Option B) 제어기 성능 비교
  - 시나리오: 동일한 기준 궤적에 대해 서로 다른 제어 파라미터/알고리즘 적용
  - 관찰 포인트: 자세 안정화 속도, 오버슈트, 정상상태 오차의 물리적 시각화
  - 정량적 결과: 제어 성능 지표와 함께 물리적 거동의 차이 제시

활용 예시 발표 시 포함 요소:
  - 시나리오 설명 (초기 조건, 입력 명령)
  - 시뮬레이터 화면 캡처 + 플랫폼 실제 동작 영상 (side-by-side 비교 권장)
  - 정량적 데이터 (각도, 각속도 시계열 그래프)
- 관찰된 특이사항 또는 한계점

시뮬레이션 예시 선정시 참고 자료: https://github.com/nasa/GTM_DesignSim

3. 시스템 구현 세부사항 파악(질의응답 대비)

  - 시스템 아키텍처 명확화 :  X-Plane, Simulink, 스튜어트 플랫폼, 그리고 FC(Flight Controller)가 실시간으로 연동되는 복잡한 구조를 가지고 있으므로, 명확한 블록 다이어그램을 제시
  - 각 기기간 통신에서 사용한 통신 프로토콜 정리
  - 신호 업데이트 주파수 식별 : 제어 주파수, X-plane 시뮬레이션 주파수, Simulink 샘플링 타임 등
  - 좌표계 변환 및 회전변환 (오일러각, 쿼터니언 등 어떤 것을 사용했는지)
  - 초기 캘리브레이션, 기준 자세는 어떻게 정의하는지

4. 구현 현황 및 성능 검증

  - 스튜어트 플랫폼의 가동 범위 : 롤, 피치, 요 범위 확인, 시뮬레이션에서 이 범위를 넘어가는 경우에는 시각화 / FC가 어떻게 처리하는지.
  - 자세각 추종 오차 확인 : X-Plane에서 특정 비행 시나리오를 실행하면서, 시뮬레이터가 출력하는 자세각(명령값)과 플랫폼이 실제로 구현한 자세각(플랫폼 제어기의 인코더 또는 FC의 IMU로 측정)을 동시에 기록한다. 이 두 신호의 차이를 계산하여 평균 오차, RMS 오차, 최대 오차, 표준편차를 구한다. (롤, 피치, 요 각각)
    - 가능한 오차 원인: (1) 통신 지연, (2) 플랫폼 제어기의 추종 성능 한계, (3) 기구학적 모델링 오차, (4) 센서 노이즈, (5) 좌표계 변환 오차 등

5. 플랫폼 활용 가치 및 확장 가능성

  - 제어알고리즘 검증 플랫폼으로서의 가치
  - 교육 및 훈련 도구로서의 가치
  - 시뮬레이터 데모 및 전시, 홍보 도구로서의 가치
  - 다양한 비행체, 차량 등 모빌리티 플랫폼으로의 확장 가능성
